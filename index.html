Pengaturan Gemini API Key

Untuk melakukan analisis secara langsung (real-time) dengan CV dan lowongan Anda sendiri, Anda membutuhkan API Key Gemini. Jika dikosongkan, aplikasi akan berjalan dalam **Mode Demo** dengan simulasi.

Cara Mendapatkan API Key Gratis:

  1. Buka Google AI Studio
  2. Masuk menggunakan akun Google Anda.
  3. Klik tombol "Get API key" di bagian atas.
  4. Klik "Create API key" lalu salin kodenya.
  5. Tempelkan kode tersebut pada kolom di atas dan klik Simpan.
